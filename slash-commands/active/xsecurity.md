# Security Analysis

Perform comprehensive security scanning with intelligent defaults. No parameters needed for basic usage.

## Usage Examples

**Basic usage (runs all security checks):**
```
/xsecurity
```

**Quick secret scan:**
```
/xsecurity secrets
```

**Dependency vulnerability check:**
```
/xsecurity deps
```
