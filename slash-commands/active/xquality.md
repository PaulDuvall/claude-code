# Code Quality Analysis

Run comprehensive code quality analysis with smart defaults. No parameters needed for basic usage.

## Usage Examples

**Basic usage (runs all checks):**
```
/xquality
```

**Quick fix common issues:**
```
/xquality fix
```

**Generate detailed report:**
```
/xquality report
```
